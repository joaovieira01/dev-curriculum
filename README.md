# 📄 Criador de Currículos Profissional

Uma aplicação web moderna e completa para criação de currículos profissionais com preview em tempo real e exportação para PDF otimizada para sistemas ATS (Applicant Tracking Systems).

## 🚀 Funcionalidades

### ✨ **Interface Intuitiva**
- **Design Responsivo**: Interface adaptável para desktop, tablet e mobile
- **Preview em Tempo Real**: Visualização instantânea das alterações
- **Modo Toggle**: Alternar entre formulário e preview para melhor experiência
- **Interface Limpa**: Design moderno com componentes shadcn/ui

### 📝 **Seções Completas do Currículo**
- **Dados Pessoais**: Nome, título, contatos e redes sociais
- **Resumo Profissional**: Descrição personalizada do perfil
- **Experiência Profissional**: Histórico completo de trabalho com datas
- **Educação**: Formação acadêmica e certificações
- **Cursos Complementares**: Certificações, workshops e cursos extras
- **Idiomas**: Múltiplos idiomas com níveis de proficiência
- **Habilidades**: Competências técnicas e comportamentais
- **Projetos**: Portfolio de projetos com tecnologias e links

### 🎯 **Recursos Avançados**
- **Compatibilidade ATS**: Estrutura otimizada para sistemas de recrutamento
- **Exportação PDF**: Download direto via navegador com formatação profissional
- **Título Dinâmico**: Nome do arquivo PDF personalizado automaticamente
- **Formulário Limpo**: Interface em branco para criação personalizada
- **Validação de Campos**: Campos obrigatórios e validação de dados
- **Gerenciamento de Estado**: Adição/remoção dinâmica de seções

### 🔧 **Funcionalidades Técnicas**
- **Impressão Otimizada**: CSS específico para impressão sem elementos desnecessários
- **Formatação Profissional**: Tipografia e espaçamento otimizados
- **Responsividade**: Layout adaptável para diferentes tamanhos de tela
- **Performance**: Carregamento rápido e interface fluida
- **Acessibilidade**: Componentes acessíveis e navegação por teclado

## 🛠️ Tecnologias Utilizadas

- **Framework**: Next.js 15 (App Router)
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS
- **Componentes**: shadcn/ui
- **Ícones**: Lucide React
- **Deployment**: Vercel

- **Preview Dinâmico**: Visualização instantânea das alterações no layout profissional

## 🚀 Como Usar

### 1. **Acesse a Aplicação**
Visite: [https://automated-cv-website.vercel.app](https://automated-cv-website.vercel.app)

### 2. **Preencha suas Informações**
- Complete os dados pessoais
- Adicione experiências profissionais
- Inclua formação acadêmica
- Liste cursos e certificações
- Especifique idiomas e níveis
- Adicione habilidades relevantes
- Inclua projetos importantes

### 3. **Visualize e Ajuste**
- Use o preview em tempo real
- Ajuste informações conforme necessário
- Verifique a formatação

### 4. **Exporte seu Currículo**
- Clique em "Baixar PDF"
- O arquivo será nomeado automaticamente como "Currículo [Seu Nome].pdf"
- Pronto para enviar para recrutadores!
